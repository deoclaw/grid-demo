Grid Demonstration

Quick Reference Table

You can review a more comprehensive list of properties on W3 Schools

Property Description Example
grid-template-columns Specifies the number of columns within a grid grid-template-columns: auto auto auto;
grid-template-rows Specifies the size of rows within a grid grid-template-rows: 200px 400px;
grid-row-start Specifies start of a grid row based on grid line numbers grid-row-start: 1;
grid-row-end Specifies end of a grid row based on grid line numbers grid-row-end: 3;
grid-row Shorthand for grid-row-end and grid-row-start grid-row: 1/5;
grid-column-start Specifies start of a grid column based on grid line numbers grid-column-start: 1;
grid-column-end Specifies end of a grid column based on grid line numbers grid-column-end: 3;
grid-column Shorthand for grid-column-end and grid-column-start grid-column: 1/5;
grid-column-gap Specifies the column gap between each box of content grid-column-gap: 30px;
grid-row-gap Specifies the row gap between each box of content grid-row-gap: 10px;
grid-gap Shorthand for grid-row-gap and grid-column-gap grid-gap: 10px 30px;
